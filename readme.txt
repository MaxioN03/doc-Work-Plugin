Для работы требуется установка NodeJS!

Сохранение первой страницы PDF-файлов в директории docs в PNG в формате 
"File.pdf"->"File.png"

Для запуска загрузить документы PDF в папку docs и запустить start.bat. Готовые файлы будут в папке docs/png

Опицонально:
В коде можно изменить имя сохраняемых файлов, изменить количество и/или номер сохраняемых страниц.
